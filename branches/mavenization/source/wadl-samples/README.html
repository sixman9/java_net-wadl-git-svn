Yahoo News Search Sample

A sample project that uses the wadl2java tool to create stubs for the Yahoo News Search Service. Includes a simple main method that uses the generated stubs to query for the latest Sun and Java news.

To build and run the project simply type:

ant run

Notes

  • The wadl2java tool is run during the build as an ant task, see the build.xml file and the wadl2java documentation for full details.
  • Generated code is placed in a separate source directory (gen-src) by the build.
  • YahooSearch.wadl is the top-level application description file for the Yahoo News Search service. It pulls in NewsSearchResponse.xsd and NewsSearchError.xsd to describe the format of the XML that the service generates.
  • The binding.xjb file is also included by YahooSearch.wadl. It is a JAXB customization file that modifies the default XML to Java mapping provided by JAXB.
